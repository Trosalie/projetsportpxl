\doxysubsection{Detailed Description}
Service de gestion de l\textquotesingle{}API Pennylane. 

Cette classe fournit une interface pour interagir avec l\textquotesingle{}API externe Pennylane. Elle gère la création de factures, la récupération des données clients et la synchronisation des factures entre Pennylane et la base de données locale.
