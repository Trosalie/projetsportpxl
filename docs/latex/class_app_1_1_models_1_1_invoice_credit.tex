\doxysubsection{Detailed Description}
Modèle représentant une facture d\textquotesingle{}achat de crédits. 

Cette classe gère les factures émises lors de l\textquotesingle{}achat de crédits par un photographe. Les crédits permettent d\textquotesingle{}utiliser les services de la plateforme Sport\+Pxl.
