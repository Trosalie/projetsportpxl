\doxysubsection{Detailed Description}
Service de journalisation des actions utilisateurs. 

Cette classe gère l\textquotesingle{}enregistrement des actions effectuées par les utilisateurs dans la base de données. Elle utilise un cache interne pour optimiser les performances lors de la recherche des IDs d\textquotesingle{}actions.
