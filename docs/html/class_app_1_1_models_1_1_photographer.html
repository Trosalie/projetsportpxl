Detailed Description

Modèle représentant un photographe dans la plateforme BoardPxl.

Cette classe gère les informations des photographes utilisant la plateforme. Elle hérite de Authenticatable pour la gestion de l'authentification et utilise HasApiTokens pour l'authentification via Sanctum.
